# Vehicle Insights App — Sprint 5 Project

This project was developed as part of the TripleTen Data Analysis bootcamp.  
The goal is to build a Streamlit web application to visualize and explore data from the `vehicles_us.csv` dataset.

## Features
- Histogram of the `odometer` column  
- Scatter plot comparing `odometer` and `price`  
- Boxplot of the `price` column  

## Project Structure
Sprint5/
├── app.py
├── vehicles_us.csv
├── requirements.txt
├── .streamlit/
│ └── config.toml
└── README.md

## Deploy
The application URL will be added after deployment on Render.
