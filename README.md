# Projeto Sprint 5 — Dashboard de Anúncios de Veículos

Este projeto faz parte do bootcamp de Análise de Dados da TripleTen.  
O objetivo é construir um aplicativo web em Streamlit para visualizar dados contidos no arquivo `vehicles_us.csv`.

## Funcionalidades
- Histograma da coluna `odometer`
- Gráfico de dispersão relacionando `odometer` e `price`
- Boxplot da coluna `price`
